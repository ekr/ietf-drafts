Abstract

This document describes the use of the Advanced Encryption Standard (AES) Counter Mode for use as a Transport Layer Security (TLS) confidentiality mechanism.

2.2. DTLS

DTLS can't treat this as a very long cipher stream. Basic idea is to use an implicit per-packet IV. That IV is derived by taking the initial IV and then adding the sequence number times some basic block size. There are two major possibilities here: (1) Use the global maximum record size (2^14) or (2) use the per-connection record size as negotiated with TLS extensions. The second seems unnecessarily clever.
